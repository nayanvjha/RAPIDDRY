## MVP Breakdown
To orchestrate this experience, we are building four interconnected systems:

1. **Customer Mobile App (iOS + Android):** The growth engine. Lightning-fast OTP login, a clean service catalog, slot booking, Razorpay payment integration, a 5-stage order tracking system, and real-time push notifications.
2. **Delivery Agent App (iOS + Android):** The operations engine. Google Maps navigation routing, mandatory photo capture for accountability, real-time status updates, and a transparent earnings dashboard.
3. **Laundry Partner Panel:** Tablet-optimized and designed specifically for non-tech laundry shop owners. Simple, intuitive point-of-sale functionality to mark orders complete.
4. **Admin Web Dashboard:** The command center. Complete order management, manual or auto agent assignment, granular analytics, coupon and discount control, and CSV exports for accounting.

*Backend Architecture:* Node.js REST APIs, a robust PostgreSQL database, Firebase Auth, FCM for push notifications, AWS S3 for media storage, and reliable cloud hosting. 415 hours of precise, enterprise-grade engineering.
